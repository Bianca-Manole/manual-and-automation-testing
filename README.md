# manual-and-automation-testing

This repository contains materials and resources for manual and automated testing.

## Structure

```
manual-and-automation-testing/
├── manual-testing/       # Test cases, bug reports, checklists
└── automation/           # XPath locators, selectors, automation scripts
```

## Contents

### 📋 Manual Testing
Documents for manual testing: test cases, bug reports, checklists, and test plans.

### ⚙️ Automation
XPath selectors, CSS selectors, and other resources used in test automation.
